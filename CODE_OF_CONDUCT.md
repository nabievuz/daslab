# Code of Conduct

This project adopts the Contributor Covenant, version 2.1.
The full text is at https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Contributing

All contributions follow the one-issue = one-branch = one-PR = green-CI workflow
described in [CONTRIBUTING.md](CONTRIBUTING.md).

## Reporting concerns

For Code of Conduct violations, contact a maintainer directly or open a private
GitHub Security Advisory on this repository.

For security vulnerabilities, do **not** open a public issue — follow the
responsible-disclosure process in [SECURITY.md](SECURITY.md).
