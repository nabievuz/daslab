# RFCs

Engineering Requests for Comment. Open an RFC here for any decision with material reversal cost (new language, new datastore, new framework, cross-cutting library, deployment target change).

Workflow:

1. Copy `_template.md` to `NNNN-short-slug.md`.
2. Discuss in PR.
3. On acceptance, the CTO records the decision as an ADR in [`../../docs/adr/`](../../docs/adr/) and links the RFC.
